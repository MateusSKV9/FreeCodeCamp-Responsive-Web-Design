Introdução

JavaScript é uma linguagem de script orientada a objetos e multiplataforma. Reconhecida por sua leveza, é utilizada tanto no lado cliente quanto servidor para controle programático do ambiente.

Biblioteca Padrão e Extensões

JavaScript oferece uma biblioteca padrão com objetos como Array, Date e Math. Além disso, pode ser estendida com objetos adicionais para manipulação do DOM no lado cliente e interações servidor-cliente.

Conhecimentos Prévios Necessários

Espera-se que o usuário tenha:

  • Compreensão geral da Internet e WWW.
  • Conhecimento sólido de HTML.
  • Experiência básica em programação.
Diferenças entre JavaScript e Java

Apesar de algumas semelhanças superficiais, JavaScript difere fundamentalmente de Java. Enquanto JavaScript é dinâmico e não requer tipagem estática, Java é estritamente tipado e segue um modelo de classes.

Hello World em JavaScript

Para começar, utilize o Scratchpad para escrever seu primeiro código JavaScript:

function greetMe(yourName) { alert("Hello " + yourName); } greetMe("World");
Variáveis em JavaScript

Variáveis em JavaScript são declaradas com var para escopo global ou local, e com let para escopo de bloco.

var linguagem = "Protuguês";
Escopo de Variáveis

Antes do ECMAScript 2015, o escopo de variáveis em JavaScript era limitado a funções ou ao escopo global. Com let, o escopo de bloco é introduzido.

let nome = "Mateus";
Constantes

Use const para criar constantes somente leitura. Elas seguem as regras de escopo de variáveis de bloco.

const PI = 3.14;
Tipos de Dados

JavaScript suporta seis tipos primitivos:

  • Boolean
  • null
  • undefined
  • Number
  • String
  • Symbol
  • Objetos
Declaração if_else

Use if para executar um bloco de código se uma condição for verdadeira, e else para código alternativo se não for.

if(x===1) { console.log("x == 1"); } else { console.log("x =! 1"); }
Declaração while

while executa um bloco de código enquanto uma condição específica for verdadeira.

Declarações de Funções

Defina funções com function, seguido pelo nome da função e argumentos. As funções podem retornar valores usando return.
